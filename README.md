# POLICE ARMORY

### Requirements
- progressBar


### Installation
1) Drag & drop the folder into your `resources` server folder.
2) Configure the config file to your liking.
3) Import esx_PoliceArmory.sql into your database
4) Add `start esx_poliecArmory` to your server config.
